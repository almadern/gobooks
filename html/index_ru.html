Поиск и настройка осуществляются при помощи предоставленных выше полей

Каждое поле осуществялет поиск по своему параметру:

  • Поиск - имеет 3 поля по которым будет производиться поиск
    • Автор - поиск по автору книги. Пример: Айзек Азимов
    • Название - поиск по названию книги. Пример: Мечтаю ли андроиды об электроовцах
    • Серия - поиск по серии книг. Пример: Песнь льда и пламени
  • Формат книги - В каком формате будет предоставленна книга
  • Язык книги - На каком языке будут предоставлены книги

После нажатия на кнопку "Искать" старница может быть пустой какое-то время.
Это нормальное поведение - сервер обробатывает данные, и выдаст вам их как процесс поиска будет закончен

ВАЖНО : Поиск может не найти книг, даже если вы знаете, что они должны быть.
     Если это произошло попробуйте изменить поисковый запрос:

  • Указать альтернативное значение для названия\серии\автора
  • Выполнить поиск на вашем языке
