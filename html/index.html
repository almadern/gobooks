Search and configuration are carried out using the fields provided above

Each field searches by its own parameter:

  • Search - has 3 fields that will be used to search
    • Author - search by book author. Example: Isaac Asimov
    • Title - Search by book title. Example: Do Androids Dream of Electric Sheep?
    • Series - search by book series. Example: A Song of Ice and Fire
  • Output book format - In what format will the book be provided
  • Language - In what language will the books be provided?

After clicking on the "Search" button, the page may be empty for some time.
This is normal behavior - the server processes the data and will return it to you when the search process is completed

IMPORTANT : A search may not find books even if you know there should be some.
     If this happens, try changing your search query:

  • Specify an alternative value for title\series\author
  • Search in your language
